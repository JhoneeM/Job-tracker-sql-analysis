# Job Tracker SQL Analysis

A SQL-based data analysis project focused on extracting insights from job application data, including application trends, success rates, and pipeline performance.

## 📊 Overview
This project uses advanced SQL queries to analyze job application data and generate meaningful insights related to application outcomes and timelines.

## 🔍 Key Analyses
- Application trends over time
- Success rates (applications → interviews → offers)
- Status breakdown (Applied, Interview, Offer, Rejected)
- Most frequently applied-to companies
- Average response time for applications

## 🧠 SQL Concepts Used
- Aggregations (COUNT, AVG)
- GROUP BY & ORDER BY
- Date functions (DATE_TRUNC, intervals)
- Filtering with WHERE clauses
- Sorting and limiting results

## 📁 Files
- queries.sql → Contains all analysis queries

## 💡 Use Case
This project demonstrates how SQL can be used to analyze real-world data and support decision-making, such as identifying which application strategies lead to better outcomes.

## 🛠 Tech Stack
- PostgreSQL
- SQL

Built to showcase data analysis skills using SQL, including writing efficient queries and deriving actionable insights from structured data.
